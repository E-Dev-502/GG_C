How to Use



Using in an attribute

<a style="width: (fake-tag-s) 50*2 (fake-tag-e)">


Output

<a style="width: {-50*2-}">


Using in Tag

<p>(fake-tag-s) 50*2 (fake-tag-e)</p> or

<style> p { width : (fake-tag-s) 50*2 (fake-tag-e)%; } </style>


Output

<p>{- 50*2 -}</p> or

<style> p { width : {- 50*2 -}%; } </style>


Runing a Function

<p>(fake-tag-s)"Hello!".toUpperCase()(fake-tag-e)</p>


Output

<p>{-"Hello!".toUpperCase("Hello!")-}</p>
